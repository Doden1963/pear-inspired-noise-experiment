# Software (Python / Jupyter)

This folder contains Jupyter notebooks used to analyze noise data generated by the Teensy 4.1 system.

The analysis focuses on bit extraction, statistical evaluation, and visualization of noise behavior.

---

## Files

* `lsb_ratio_1M_samples.ipynb`
  Calculates the ratio between 0 and 1 bits from a large sample set (e.g. 1,000,000 samples)

* `pear_experiment_15000hz_von_neumann.ipynb`
  Main experiment using high sample rate data and Von Neumann debiasing

* `historical_histograms.ipynb`
  Displays histograms of accumulated data to visualize statistical distributions

* `teensy_samples_150.ipynb`
  Inspection of raw data samples from the Teensy device

* `lowpass_audio_analysis.ipynb`
  Analysis of filtered signals and optional audio output

---

## Methods

The notebooks implement several key steps:

* Conversion of analog noise into digital bits
* Bit extraction using LSB and derived methods
* Optional debiasing (e.g. Von Neumann)
* Statistical evaluation using z-score and p-values
* Visualization using histograms and plots

---

## Data Source

* Data is collected from the Teensy 4.1 via serial communication
* Typical sampling rates include high-frequency acquisition (e.g. ~15,000 samples/sec)

---

## Notes

* Results should be interpreted statistically
* Random fluctuations are expected in finite samples
* Hardware setup (breadboard vs PCB) can influence results
* Noise characteristics depend on analog amplification and environment

---

## Usage

Open the notebooks using Jupyter:

```id="run_nb"
jupyter notebook
```

or

```id="run_nb2"
jupyter lab
```

Then run the cells step by step.

---

## Purpose

The goal of this software is to analyze whether statistical deviations from randomness occur under different experimental conditions (e.g. UP, DOWN, NONE).
